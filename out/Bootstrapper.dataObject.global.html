Namespace: global

Bootstrapper.dataObject.global

The global scope of the data object. Can be queried with Bootstrapper.dataObject.addData and queried with Bootstrapper.dataObject.getData. These functions can only query up to 3 layers deep, otherwise the data must be retrieved directly from the object. This scope contains all of the other scopes maintained for the Airline site.
