(static) getData(key, layer, scope) → {Dynamic}

This function works in backward specifity from the order of parameters, such that the Bootstrapper.dataObject function is structured in such a way that Bootstrapper.dataObject[scope][layer][key] is the value returned by Bootstrapper.dataObject.getData(key, layer, scope). Therefore for any string set of (key, layer, scope) strings, 
Bootstrapper.dataObject.getData(key, layer, scope) === Bootstrapper.dataObject[scope][layer][key]

For example: 

Bootstrapper.dataObject.getData('partnerID', 'adara', 'global') === Bootstrapper.dataObject['global']['adara']['partnerID'];
On Airline Staging, both the function and the data lookup return the integer 1020. Feel free to test for yourself in the javascript console on the page If the tuple cannot be found, return the empty string ("")
Parameters:
Name Type Description
key String the key whose value we want
layer String the Data Layer from which we want to grab the content
scope String the top-level scope from which the data comes from
